#  Microsoft 365 Groups and Shared Mailbox Management Lab

This lab demonstrates how **Microsoft 365 Groups** and **Shared Mailboxes** are created and managed, including assigning owners and members, configuring automatic replies, and validating shared mailbox access and email communication.

---

##  Step-by-Step Implementation

### Step 1: Create a Microsoft 365 Group
- Navigated to **Home → Teams & groups → Active teams & groups**
- Clicked **Add a Microsoft 365 group**
- Entered group name: **Sales Department**
